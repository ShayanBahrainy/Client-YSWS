# Client YSWS
This is a YSWS website I made for the 2026 Hack Club internship applications. The premise is that you create a game client following some specs, and then you play the games together because they'll be compatible.
I designed the site mostly in figma, and then exported the slides as SVGs. The rest of website just plays the SVGs like a slideshow.

# Running
Load index.html in your web browser to run it locally.
